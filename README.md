# Master Agent

Автономный ИИ-помощник для Android на базе Gemini API Free Tier.

## Возможности
- Голосовая активация через триггер-слово "Мастер"
- Чтение структуры экрана через Accessibility Service
- Клики, свайпы и ввод текста на любом экране
- Озвучка сообщений и диалог через Text-To-Speech

## Сборка
Проект автоматически собирается в облаке через GitHub Actions при коммите в ветку `main`. Скачать APK-файл можно во вкладке "Actions" репозитория.
